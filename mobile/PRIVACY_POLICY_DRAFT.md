# Política de Privacidade — Rascunho

Este documento é um rascunho operacional para revisão antes da publicação do
aplicativo ColaboraEdu Família.

## 1. Quem somos

O ColaboraEdu Família é um aplicativo de acompanhamento escolar usado por
alunos e responsáveis de instituições que utilizam a plataforma ColaboraEdu.

## 2. Dados tratados

O aplicativo pode exibir dados escolares vinculados ao estudante, incluindo:

- nome do aluno;
- matrícula;
- turma e turno;
- notas e boletim;
- faltas;
- ocorrências escolares;
- comunicados da escola;
- dados básicos da conta de acesso.

## 3. Finalidade

Os dados são usados para permitir que alunos e responsáveis acompanhem a vida
escolar, recebam comunicados e consultem informações acadêmicas fornecidas pela
escola.

## 4. Login e segurança

O acesso é protegido por usuário e senha. Quando uma senha temporária é gerada,
o aplicativo solicita a troca no primeiro acesso.

## 5. Compartilhamento

O aplicativo não vende dados pessoais. Os dados são exibidos somente ao usuário
autenticado e autorizado pela escola.

## 6. Armazenamento no dispositivo

O aplicativo pode armazenar tokens de sessão no armazenamento seguro do
dispositivo para manter o usuário autenticado. O usuário pode encerrar a sessão
a qualquer momento pelo botão de sair.

## 7. Direitos do titular

Solicitações de correção, acesso ou exclusão de dados escolares devem ser
encaminhadas à escola responsável pelo cadastro do estudante.
